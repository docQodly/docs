Version: 1.1.0

1.0.0 - beta.6

11/05/2024​

Qodly Studio​

What's new

  • New onPrivilegeChange Event: Ensures elements display based on user privileges.

  • Sanity check Update: Raises an error if a shared selection attribute is dropped over a matrix bound to a local selection.

  • Binding onHttpGet Functions: Enables binding functions for HTTP GET requests with 4D.OutgoingMessage, supporting flexible operations like file viewing or downloads.

  • Toggle Inherited Permissions: Allows visibility toggling to show only explicitly defined permissions or include inherited ones by default in the privileges interface.

  • Default State Override Warning: Displays a warning icon when the Default State will be overridden by an unmet Conditional State condition, with a tooltip explaining the override.

Qodly Server​

What's New

  • Session Singletons: Added support for Session singletons, including the new .isSessionSingleton class property.

  • onHttpGet Function Keyword: Allows defining singleton or ORDA functions accessible through REST requests using the HTTP GET verb.

  • New 4D.OutgoingMessage Class: Enables the REST server to return any web content, enhancing response flexibility.
