## Overview

Qodly Studio utilizes modal dialogs to promptly alert users about potential errors or conflicts requiring resolution. Below are examples of situations where these notifications may occur:

| Error Modal                                                                                                                            | Trigger Condition                                                      | Reason                                                                                                                                                                      | Solution                                                                                         |
| :------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------- |
| **Permissions Errors:** Cannot find member function 'Project.create'                                                                   | When trying to invoke 'Project.create' without definition.             | This error indicates that the function within the Project class no longer exists.                                                                                           | Define a function with the same name or remove the permission if unnecessary.                    |
| **Permissions Errors:** Cannot find dataclass 'Project'.                                                                               | When trying to access the dataclass 'Project' and give it permissions. | This error suggest that the 'Project' dataclass is missing.                                                                                                                 | Define the 'Project' dataclass and ensure it includes an update function as needed.              |
| **Permissions Errors:** Cannot find attribute 'Project.subject'.                                                                       | when attempting to access 'Project.subject' attribute.                 | This error occurs when the 'subject' attribute is not defined within the 'Project' class.                                                                                   | Add a 'subject' attribute to the 'Project' class or verify its existence and access permissions. |
| **Error code:** 1588 (dbmg) the "attributeName" attribute is duplicated Database Engine Database Engine task 38, name: 'REST Handler:' | Save the modal changes                                                 | using an attribute name as an alias name                                                                                                                                    | change the alias name                                                                            |
| **Error fetching tab info**                                                                                                            | Click on the preview and go back to the studio                         | As expected, if you have an opened page (webform) in Qodly Studio and you stop the server of the instance, the errors are displayed once you go back to Qodly Studio's tab. | Start the server                                                                                 |
| **Cannot save file**                                                                                                                   | Save the webfrom changes                                               | You already have the "Error fetching tab info" error                                                                                                                        | Start the server                                                                                 |
